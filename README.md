#  RENTAL PROPERTY PAYMENT TRACKING SYSTEM

##  Overview
The Rental Property Payment Tracking System is designed to help property owners efficiently manage multiple rental properties, track rent payments, monitor expenses, and gain visibility into both financial and operational performance.

This system addresses the common challenges faced by landlords, especially those handling multiple tenants and properties, by providing structured data tracking and insightful reporting.

---

## Problem Statement
Property owners often face difficulties in managing:

- Tracking rent payments across multiple tenants and properties
- Monitoring irregular or delayed payments
- Calculating actual net cash flow after expenses
- Managing operational tasks like maintenance, complaints, and cleaning
- Maintaining visibility over property performance

---

##  Objectives
- Simplify rental income and expense tracking
- Provide clear financial insights (cash flow, profitability)
- Track operational responsibilities and tasks
- Enable better decision-making using reports and dashboards

---

##  Pain Points
- Manual tracking using spreadsheets leads to errors
- Lack of centralized data for multiple properties
- Difficulty in calculating real profit after expenses
- Poor visibility of operational tasks
- Missed or delayed rent payments

---

##  Key Performance Indicators (KPIs)
- Total Rental Income (Monthly/Annual)
- Occupancy Rate (%)
- Rent Collection Efficiency (%)
- Net Cash Flow
- Expense Ratio (Expenses / Income)
- Maintenance Cost per Property
- Tenant Payment Timeliness

---

##  Data Fields

### Property Details
- Property ID
- Location
- Property Type (Apartment, House, Commercial)
- Owner Name

### Tenant Details
- Tenant Name
- Contact Information
- Lease Start & End Date
- Rent Amount
- Payment Frequency (Monthly/Quarterly)

### Payment Details
- Payment Date
- Amount Paid
- Payment Status (Paid/Pending/Delayed)
- Mode of Payment

### Expense Details
- Expense Type (Tax, Maintenance, Repair)
- Amount
- Date
- Property Linked

### Operational Tasks
- Task Type (Cleaning, Repair, Complaint)
- Assigned To (Manager/Caretaker)
- Status
- Completion Date
